# 밴픽 시뮬레이터 밸런스/특성 설계 프롬프트

아래 조건으로 LoL 밴픽 시뮬레이터용 챔피언 밸런스 패치를 설계해줘.

## 목표
- 특정 조합/특성이 과도하게 승률을 독식하지 않게 조정
- 챔피언별 개성이 유지되도록 수치 조정
- 결과는 **CSV에 바로 반영 가능한 형태**로 제시

## 입력 데이터
- 챔피언 데이터: `champions.csv`
- 특성 데이터: `traits.csv`
- 최근 테스트 결과(있다면):
  - 챔피언 승률 상/하위 20
  - 조합 유형(돌진/포킹/받아치기) 승률
  - AD/AP 쏠림 구간별 승률

## 규칙
1. 과도한 버프/너프 금지
- 기본 스탯(딜/탱/CC/초중후)은 1회 수정폭 최대 ±2
- 유형 스케일(1~3)은 최대 ±1

2. 특성 보정 제한
- 쉬운 조건 특성: 총 기대값 낮게
- 어려운 조건 특성: 총 기대값 높게
- 단일 특성의 직접 승률 보정은 절대값 8 이내 권장

3. 출력 형식
- 반드시 3개 섹션으로 출력
  - `A. 문제 진단`
  - `B. 수정안(챔피언)`
  - `C. 수정안(특성)`
- `B/C`는 표 형태로 출력
  - 컬럼: `대상`, `현재값`, `제안값`, `근거`, `예상영향`

4. 최종 체크리스트
- 변경 후 예상 메타 요약 5줄
- 리스크 3개
- 추가 테스트 시나리오 5개

## 요청
현재 데이터 기준으로 1차 밸런스 패치안을 만들어줘.
